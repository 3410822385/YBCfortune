# YBCfortune #
Minecraft Plugin that does 
* /roll
* Handles Whispers
* Handles in chat Links 
* Uses Spigot annouce to send timed banners

Implements class Forturnes for rolls
The rolls are split into 2 catergories Normal(80%) and Rare(20%).
